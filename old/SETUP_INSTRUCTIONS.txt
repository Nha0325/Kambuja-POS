FTC POS - Setup Instructions
============================

DATABASE SETUP:
--------------
1. Make sure Laragon MySQL is running
2. Open Command Prompt and run:
   
   mysql -u root -p < C:\laragon\www\ftc-pos\full_fixed.sql

   (Press Enter when asked for password if root has no password)

   OR use this command:
   mysql -u root < C:\laragon\www\ftc-pos\full_fixed.sql
