# Feature Map

## ADMIN_MANAGER

- authenticate through the platform manager UI
- view platform totals
- create ADMIN accounts
- view all ADMIN accounts
- view all shops
- filter shops and reports by province and city
- view platform sales reports
- view platform stock reports
- view notifications
- manage platform settings

## ADMIN

- authenticate through the shop UI
- view shop dashboard
- edit shop profile
- manage products and product codes
- manage categories
- manage inventory and stock movements
- create and list CASHIER accounts
- view sales reports
- view customer records
- configure Telegram chat settings
- manage shop settings
- view notifications and audit logs

## CASHIER

- authenticate through the shop UI
- search products
- scan barcode or QR product codes
- manage cart quantities
- create a sale
- collect payment
- generate and print receipt
- view today's sales

## CUSTOMER_RECORD

Customer records are stored in the `customers` collection. They do not authenticate and are not represented by a login role.
