Short checklist for Windows

1. Install XAMPP / WAMP / IIS + PHP.

2. Enable curl, pdo_mysql, openssl, mbstring in php.ini.

3. Install wkhtmltopdf for Windows.

4. Add wkhtmltopdf to PATH or use full path in helpers.php.

5. Make sure shell_exec is not disabled in php.ini.

6. Ensure PDF folder is writable.

7. Test Telegram via small script.
